УСТАНОВКА:

1. Скопировать все файлы на сервер (например, в директорию test на localhost)
2. Отредактировать config.php, указав настройки коннекта к базе данных
3. Установить привилегии 777 (rwxrwxrwx) на директории upload, theme\templates, theme\templates_c 
4. Открыть через http (например, http://localhost/test/)
5. При успешном выполнении удалить файл install.php
